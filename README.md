# Shoko Toolkit

Python Toolkit for maintaining large anime libraries.

## Features

- Library Scanner
- Shoko API
- Health Report
- Hardlink Verification
- Duplicate Detection
- Cleanup
- Repair

## Installation

```bash
pip install -r requirements.txt
```

## Configuration

Copy

config.yaml.example

to

config.yaml

and edit the values.

## Usage

```bash
python main.py
```
